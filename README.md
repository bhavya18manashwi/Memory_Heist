## 🚀 About The Project

**Memory Heist** is an immersive multiplayer game where players become elite investigators tasked with analyzing complex scenes, spotting hidden clues, retaining critical information, and answering challenging AI-generated questions.

Success depends on observation, teamwork, memory retention, and quick decision-making. Every round presents a new challenge, ensuring a unique and engaging experience for players.

Whether you're competing with friends or testing your cognitive abilities, Memory Heist transforms memory training into an exciting game.

---

## ✨ Key Features

### 🎯 Observation Challenges

Carefully inspect detailed scenes and identify hidden clues before time runs out.

### 🤖 AI-Powered Question Generation

Dynamic questions generated using Gemini AI based on scene content and gameplay context.

### 👥 Real-Time Multiplayer

Create game rooms, invite friends, and compete in collaborative memory challenges.

### 💬 Team Communication

Coordinate strategies and share observations through integrated team chat.

### 🏆 Competitive Leaderboards

Earn points for accuracy and speed while climbing the rankings.

### ⚡ Fast & Responsive Experience

Modern interface with smooth interactions and real-time updates.

### 🎮 Endless Replayability

AI-generated content ensures fresh challenges in every session.

## 📦 Installation

### Clone Repository

```bash
git clone <repository-url>
cd memory-heist
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
```

### Run Development Server

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:3000
```
