# Memory Heist Operational User Manual

Welcome Agent, to the **Memory Heist** operational directive center. This manual covers interactive controls, challenge mode layouts, and score calculations.

---

## 🎮 CORE CHALLENGE SECTORS

### 👤 Objects and Coordinates
- **The Heist**: Memorize exact objects, color palettes, and positions.
- **Example**: "Red Steel Trunk next to platform 3 bench", "Gold Vintage Clock centered on the main archway".
- **Tactical Tip**: Pay attention to qualifying adjectives! A question might test whether a kettle was brass, clay, or plastic.

### 🔢 High-Speed Numbers
- **The Heist**: Spot and secure rapid digit coordinates displaying on route trackers or scoreboards.
- **Example**: `937256`.
- **Tactical Tip**: Chunk digits into pairs (93 - 72 - 56) to record them quickly in memory!

### 🔮 Pattern Matrices
- **The Heist**: Memorize a 3x3 layoutboard containing colored lab beakers, element diagrams, or stylish footwear boxes.
- **Example**: Row 2 Column 2 color Green containing a Sneaker.
- **Questions**: Test counts of specific colors, or coordinate symbols.

### 📜 Narrative Story
- **The Heist**: Read a cozy, family-friendly mini narrative describing royal Jaipuri baaraats, busy Delhi street vendors, or college libraries.
- **Example**: "Aarav arrived carrying a silver tray with 5 red envelopes, while Priya coordinated 12 dhol players to perform at exactly 07:00 PM."

## 📡 COOPERATIVE SYNC (MULTIPLAYER LOBBIES)
1. Select **Multiplayer Room** toggle in configuration lanes.
2. Click **Generate Room** to get an encrypted 6-character room code (e.g. `MH4287`).
3. Share the code with your friends so they can join instantly!
4. Once everyone toggles **Ready**, the observation board triggers.
